# Vim9 Vibe Coding vimrc 🚀

これは純粋な Vibe Coding です。

Vim と雰囲気（vibe）だけで書かれた Vim9 Script 製の vimrc で、  
正式な Vim スクリプトの経験は不要です。  
主な目的は、Vim を使っていて気持ちよく楽しい体験にすること。  
コード探索用に ChatGPT 連携も入っています。

---

## ✨ 特徴

- ✅ Vim9 Script のみ
- 🎨 見やすいダークテーマとシンタックスハイライト
- ⌨️ Smart Tab 補完（コンテキストに応じて自動インデント or 補完）
- 🤖 ChatGPT 連携
  - ビジュアルモードで選択したコードをブラウザで直接送信
- 🧠 最小限の設定で「なんか気持ちいい」を追求

---

## 🧩 機能

### 🔹 基本設定
- 行番号 / 相対行番号
- カーソル行のハイライト
- ファイル情報付きステータスライン
- 自動インデント、スマートインデント、タブ幅、検索挙動など

### 🔹 見た目
- ダーク背景
- 目に優しいシンタックスカラー
- コメント、キーワード、関数、文字列がわかりやすくハイライト

### 🔹 Smart Tab
```vim
Tab       → インデントまたは補完
Shift+Tab → 前の補完候補
Enter     → 補完確定または改行
```

### 🔹 ChatGPT 連携

1. ビジュアルモードでコードを選択  
2. `<leader>g` を押す  
3. ブラウザで ChatGPT が開く  

自動で以下のプロンプトが含まれます：

以下のコードについて：
1. 何をしているコードかの解説
2. 改善できる点
3. バグや危険な箇所の可能性

### 🛠 必要環境

- Vim 9.0 以上
- vim9script 有効
- python3 インストール済み
- 対応 OS:
  - macOS (`open`)
  - Linux (`xdg-open`)
  - Windows (`cmd /c start`)

> **❌ 注意:** Neovim には非対応（Vim9 Script 必須）

### 📦 使い方

1. リポジトリをクローン
2. ~/.vimrc（または好みの設定パス）にコピー
3. Vim を起動
4. Vibecoding を楽しむ ✨

### ⚠️ 注意点

- この vimrc は実験的で、雰囲気だけで書かれています
- 大規模運用や堅牢性は考慮していません
- ChatGPT の URL はサービス変更で壊れる可能性あり
- 自己責任で使用してください
