MVrc は
「シンプルで、とっつきやすい Vim」を目指して作られた
プラグイン不要の Vim9 設定です。

---

## 🚀 はじめよう

### ① Vim 9.x を確認

```
vim --version
```

Vim 9.x であることを確認してください。

---

### ② 設定ファイルに貼り付け

#### macOS / Linux

```
~/.vimrc
```

#### Windows

```
~/vimfiles/_vimrc
```

上記ファイルに MVrc の内容をコピー＆ペースト。

---

### ③ 再起動

Vim を再起動すれば完了です。

プラグインのインストールは不要です。

---

## なぜ MVrc を作ったのか

Vim を理想の環境にしようとすると、

* プラグイン選びで迷う
* 依存関係で壊れる
* 設定が複雑になりすぎる
* 何が動いているのかわからなくなる

そうして挫折してしまう人は意外と多い。

だったら、

**標準 Vim9script だけで完結する構成にすればいい。**

MVrc はその発想から生まれました。

---

## ✨ コンセプト

* 誰でも使いやすい
* 迷わない構成
* 読めば理解できる
* プラグイン不要
* モダンで統一感のある見た目

MVrc は「最強の Vim」ではありません。

**安心して使える Vim**です。

---

## 🧩 主な機能

### 🎨 モダンダークテーマ

統一されたダーク配色。
ステータスライン・ポップアップ・ファイルツリーも一貫設計。

---

### 🌿 Git ブランチ表示

* 現在のブランチを表示
* detached HEAD 対応
* 保存や移動時に自動更新

---

### 🌲 ファイルツリー（標準 netrw）

開く：

```
\e
```

* 左側に縦分割
* 行番号なし
* バナーなし
* テーマ統一済み

閉じる：

```
\x
```

※トグルではなく明示的に閉じる設計です。

---

### 🤖 ChatGPT 連携（詳細）

MVrc には、ブラウザ版 ChatGPT を開く軽量連携機能があります。

#### ① ビジュアル選択して送信

1. コードを選択

```
v
```

2. 範囲を選ぶ
3. 送信

```
\g
```

選択したコードは、以下のテンプレ付きで送信されます：

* 何をしているコードか
* 改善点
* バグの可能性

ブラウザが自動で開きます。

---

#### ② コマンドで直接送信

```
:Chat <質問内容>
```

例：

```
:Chat このアルゴリズムを高速化する方法は？
```

---

#### ③ 範囲指定コマンド

```
:ChatGPT
```

ビジュアルモードで選択後に実行可能。

---

#### 必要条件

* python3 がインストールされていること
* インターネット接続

---

### 🪟 ウィンドウ管理

#### 右に分割

```
\ + 右矢印キー
```

#### 下にターミナルを開く

```
\t
```

12行サイズで開きます。

#### 保存して閉じる

```
\x
```

未保存なら自動保存して閉じます。

---

### 🧠 スマート補完

* `<Tab>`：文脈によって補完 or インデント
* `<S-Tab>`：逆方向移動
* `<CR>`：ポップアップ確定

## Vim 基本操作

MVrc は Vim の基本操作を前提にしています。

---

### モード

挿入モード：

```
i
```

ノーマルモードへ戻る：

```
Esc
```

ビジュアルモード：

```
v
```

---

### 保存と終了

保存：

```
:w
```

終了：

```
:q
```

保存して終了：

```
:wq
```

保存せず終了：

```
:q!
```

---

### 画面移動（分割時）

```
Ctrl + w + ←
Ctrl + w + →
Ctrl + w + ↑
Ctrl + w + ↓
```

---

## ⌨️ MVrc 独自キーマップ一覧

| キー               | 機能              |
| ---------------- | --------------- |
| `\e`             | ファイルツリーを開く      |
| `\g`             | 選択範囲をChatGPTへ送信 |
| `:Chat <prompt>` | 任意テキストを送信       |
| `\t`             | ターミナルを開く        |
| `\x`             | 保存して閉じる         |
| `\ + →`          | 右に分割            |
| `\y`             | クリップボードへコピー     |
| `\p`             | クリップボードから貼り付け   |
